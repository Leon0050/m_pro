# 常用任务助手 (Skill2)

这个 Skill 专门用于处理用户平时最频繁的操作流程。

## 1. 每日资讯获取 (News Briefing)
当用户询问今日新闻时，执行以下逻辑：
- **AI 资讯**：搜索最新的 AI 技术突破、大模型发布和行业动态。
- **金融股市**：搜索全球指数（S&P 500, NASDAQ, FTSE）、重大财经事件（如政府停摆风险）及科技股表现。
- **格式要求**：使用清晰的标题、简要摘要、关键点和原始链接。

## 2. Skill 部署 (Skill Deployment)
当用户提供 GitHub 链接要求部署 Skill 时：
- 克隆仓库到临时目录 `/tmp`。
- 识别其中的 `skills/` 或核心文件夹。
- 将目标内容复制到用户的 `llm/trae` 或指定目录。
- 清理临时文件。

## 3. 工作区整理 (Workspace Organization)
- 保持 `llm/` 目录结构整洁。
- 自动化创建必要的子目录（如 `agent/skills`）。
- 按照任务属性归类文件。

## 触发词示例
- "帮我获取今日的金融股市新闻"
- "把这个 GitHub 仓库里的 skill 部署一下"
- "帮我整理一下 llm 文件夹"
